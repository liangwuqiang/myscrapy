正则表达式# [] 集合


[] 集合：


可以匹配集合里面任意一个的意思


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

    # [] 集合

    print(re.search(r'0[xX]([0-9A-Fa-f]+{6})', 'The hex value is 0xFF03D6').groups())   //我们把十六进制的值给它提取出来

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，正确的匹配了。
