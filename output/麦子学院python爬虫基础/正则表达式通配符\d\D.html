正则表达式通配符\d\D


\d\D：


（\d表示匹配一个数字，\D表示匹配一个非数字）


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # \d\D

    print(re.search(r'(\d{3}-\d{4}-\d{4})', 'The Phone Number is 138-2231-2398').groups())

if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果



修改代码：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # \d\D

    print(re.search(r'(\d{3}-\d{4}-\d{4})', 'The Phone Number is 1380-2231-2398').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，还是会匹配出来，而且这个时候是从380开始匹配，因为这里面是search，所以它会去往下找，所以这个字符串还是符合这个模式。



为了保证输入电话号码的合法性，所以我们这里要修改代码：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # \d\D

    print(re.search(r'\b(\d{3}-\d{4}-\d{4})\b','The Phone Number is 138-2231-2398').groups())   //\b实际上它不匹配任何东西，但是它会去匹配一个单词的开头或者结尾，也就是说，Number的前一部分和它的后一部分就是它的开头和结尾，还有电话号码的前一部分和它的后一部分也是它的开头和结尾


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，匹配出了正确的电话号码。

在写电话号码的时候采用不一样的连字符


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # \d\D

    print(re.search(r'\b(\d{3}\D\d{4}\D\d{4})\b','The Phone Number is 1380 2231=2398').groups())   //这里就不能使用-了，采用\D


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，这样也能正确的匹配电话号码。
