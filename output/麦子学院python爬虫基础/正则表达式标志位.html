正则表达式标志位


re.IGNORECASE：


标志位，忽略大小写


在http里面，我们的标志头是忽略大小写的


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # re.I/re.IGNORECASE

    print(re.match(r'(name)\s*:\s*(\w+)', 'Name: Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，这样就能把名字和它的值给它匹配下来。



因为是大小写，所以我这里把它写为大写看看。


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # re.I/re.IGNORECASE

    print(re.match(r'(name)\s*:\s*(\w+)', 'NAME: Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，就会匹配失败。



如果要忽略大小写的话。


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

     # re.I/re.IGNORECASE

print(re.match(r'(name)\s*:\s*(\w+)', 'NAME: Joey, re.IGNORECASE').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，这样就可以正确的匹配出来了。
