正则表达式通配符\w\W


\w\W:


\w表示匹配一个字符，包括[a-zA-Z0-9_]


找出它的标题和值


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

    # \w\W: [a-zA-Z0-9_]

    print(re.match(r'(\w+)\s*:\s*(\w+)', 'Name: Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，这样就可以把标题和结果给它匹配出来。



那如果我们这里加个=会怎么样呢？


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

    # \w\W: [a-zA-Z0-9_]

    print(re.match(r'(\w+)\s*:\s*(\w+)', 'Name:= Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，匹配不成功，因为\w+会去匹配字符，它不包含=号。
