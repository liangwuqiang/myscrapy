正则表达式之greedy


greedy：（贪婪）


找出html前面的标签<H1>


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # greedy

    s = '<H1>title</H1>'

    print(re.match(r'<.*>', s).group())  //.*一到多个任意的字符

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，是全部匹配了，并不是指匹配了<H1>，这个表达式它默认的是使用贪婪模式匹配的，它的>符号默认匹配到了最后。



修改代码：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # greedy

    s = '<H1>title</H1>'

    print(re.match(r'<(.*)>', s).group(1))  //.*一到多个任意的字符

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，它把中间的字符串H1>title</H1都匹配成了任意字符。



non-greedy：（非贪婪）


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # non-greedy

    s = '<H1>title</H1>'

    print(re.match(r'<.*?>', s).group())   //?表示使用非贪婪的模式去匹配任意字符

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果



修改代码：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # greedy

    s = '<H1>title</H1'

    print(re.match(r'<(.*)>', s).group(1))   //?表示使用非贪婪的模式去匹配任意字符

   

if __name__ == '__main__':

运行结果，虽然它还是采用的默认的贪婪的模式去匹配，但是它后面已经没有>符合了，它最后的>符号是在<H1>。



{m}：（次数）


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # {m}

    print(re.match(r'abb', 'abbccdd').group())

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，匹配2个b。



另一种表示方法：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # {m}

    print(re.match(r'ab{2}', 'abbccdd').group())  //匹配2个b

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果

# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # {m}

    print(re.match(r'ab{2,4}', 'abbbbbb').group())  //匹配2个到4个b

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，2个到4个b的时候，它会尽量去匹配4个4，这个是贪婪的写法。



如果我不贪婪呢？修改代码：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # {m}

    print(re.match(r'ab{2,4}?', 'abbbbbb').group())  //匹配2个到4个b

   

if __name__ == '__main__':

re_pattern_syntax()


运行结果，这种情况下会尽量的匹配少的。



转义字符 \ 用来匹配特殊字符:


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax():

   # 转义字符 \ 用来匹配特殊字符

   print(re.search(r'\$', 'The price is $9.00').groups())   //匹配$

   

if __name__ == '__main__':

re_pattern_syntax()
