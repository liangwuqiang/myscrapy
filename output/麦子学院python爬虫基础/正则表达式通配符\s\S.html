正则表达式通配符\s\S


\s\S:


\s表示匹配一个空格，\S表示匹配一个非空格，空格是一个广义的符号，包括[ \t\n\r\f\v] \f: 换页 \v: 垂直制表


提取名字：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

   # \s\S: [ \t\n\r\f\v] \f: 换页 \v: 垂直制表

       print(re.match(r'Name:\s*:\s*([a-zA-Z]+)', 'Name: \tJoey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果



如果你在这里多几个空格也是可以的，修改代码

运行结果，这样就匹配不到。



如果修改为search后：


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

   # \s\S: [ \t\n\r\f\v] \f: 换页 \v: 垂直制表

       print(re.search(r'Name:\s*:\s*([a-zA-Z]+)', 'FirstName: \t   Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()


运行结果，就可以匹配了。



如果我们放宽一点标准，也就是标题部分可以是任意的非空白符都可以。


# -*- coding: utf-8 -*-

import re


def re_pattern_syntax_meta_char():

   # \s\S: [ \t\n\r\f\v] \f: 换页 \v: 垂直制表

       print(re.search(r'\S*:\s*([a-zA-Z]+)', 'FirstName: \t   Joey').groups())


if __name__ == '__main__':

re_pattern_syntax_meta_char()
