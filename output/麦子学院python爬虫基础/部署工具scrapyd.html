部署工具scrapyd


Srapyd是什么


1、Scrapyd 是一个部署和运行scrapy spiders的应用。它可以使用JSON API部署工程并控制spider。

2、scrapyd可以管理多个工程，并且每个工程都可以有多个版本，但只运行最新的那个版本。

 

安装Scrapyd


1、前提


（老师机器的配置，其他版本组合未测）

Scrapy  : 1.0.3

Twisted : 15.4.0

Python  : 2.7.10


2、安装


(前提是必须安装了pip)

pip install scrapyd


3、测试


在命令行中执行：

scrapyd

启动scrapyd服务,会打印日志信息；
